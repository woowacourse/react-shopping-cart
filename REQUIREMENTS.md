## 도메인

## UI

- [ ] 장바구니 상품 목록을 체크박스와 함께 보여준다.
- [ ] 각 상품의 스테퍼로 수량을 조절할 수 있다.
- [ ] 가격에 수량을 곱한 합계 값을 보여준다.
- [ ] 각 상품의 휴지통 버튼을 클릭시 해당 상품이 장바구니에서 없어진다.
- [ ] '전체선택' 체크박스 클릭시 장바구니에 있는 모든 상품의 체크박스가 체크된다.
- [ ] 체크박스가 체크된 상품과 전체상품 수가 표시된다. (선택된상품/전체상품)
- [ ] 선택된 상품의 합계 가격을 보여준다.
- [ ] '전체삭제' 버튼 클릭시 장바구니에서 모든 상품이 없어진다.
- [ ] 상품이 없을 때 대체 컴포넌트를 보여준다.
- [ ] 오류상황 처리하기 -[ ] api통신이 실패한 경우, -[ ] 화면에 보여줄 데이타가 오지 않은 경우
